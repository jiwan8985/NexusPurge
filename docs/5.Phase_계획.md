# UploadTool Phase 계획

## 개요

UploadTool의 Phase 계획은 **기존 툴 대체 기반 정리 → S3 기능 재현 → CDN 구조 확장 → 인증/로그 구조 준비 → 고객사 확인 후 실제 API 구현 → 후속 고도화** 순서로 진행합니다.

전체 개발은 신규 기능 확장이 아니라 **기존 CDN 업로드/Purge 툴의 기능 재현**을 1차 목표로 합니다.
