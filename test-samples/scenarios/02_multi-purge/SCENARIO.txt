시나리오 02 — 다중 파일 선택 Purge
================================================================

목적: 여러 파일을 동시에 선택하여 배치 Purge가 올바르게 동작하는지 확인

[순서]
1. 폴더 내 모든 파일(file-01~05, no-purge.txt) 업로드
2. 원격 패널에서 file-01~05.txt 5개만 선택 (no-purge.txt 제외)
3. [선택 Purge] 실행
4. PurgeDialog에서 5개 경로 목록 확인

[검증]
file-01~05.txt → X-Cache: Miss (Purge 적용됨)
no-purge.txt   → X-Cache: Hit  (캐시 유지됨)

[기대 결과]
- PurgeDialog 경로 미리보기에 5개 경로 표시
- Purge API 호출 1회 (5개 경로 일괄)
- no-purge.txt 캐시 영향 없음
