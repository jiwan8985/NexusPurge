시나리오 06 — 대량 배치 Purge (경고 임계값 테스트)
================================================================

목적: 다수 파일 Purge 시 경고 팝업 및 배치 처리 동작 확인
      현재 설정: 경고 기준 1,000개 / 배치 크기 1,000개

[파일 구성]
batch-file-01.txt ~ batch-file-20.txt (총 20개)

[테스트 A — 정상 범위 배치 Purge]
1. 20개 파일 모두 업로드
2. 전체 선택 → [선택 Purge]
3. PurgeDialog에서 20개 경로 미리보기 확인
   (PREVIEW_MAX=8 이므로 8개 표시 + "+12개 더")
4. [Purge 실행] 후 LogPanel에서 결과 확인

[테스트 B — 경고 임계값 테스트]
설정 → 전송 성능 → Purge 경고 기준을 10으로 낮춤
1. 20개 파일 선택 → [선택 Purge]
2. PurgeDialog에 주황 경고 배너 표시 확인
   "10개 이상 Purge — 처리 시간이 길어질 수 있습니다."
3. [Purge 실행] 후 정상 동작 확인

[테스트 C — 배치 크기 초과 테스트]
설정 → 전송 성능 → Purge 배치 크기를 5로 낮춤
1. 20개 파일 선택 → [선택 Purge]
2. PurgeDialog에 빨간 오류 배너 표시 확인
   "배치 크기(5개) 이상 Purge는 CDN 제한에 도달할 수 있습니다."

[기대 결과]
A: 정상 Purge 20건 처리 ✓
B: 주황 경고 배너 표시 ✓
C: 빨간 한도 초과 배너 표시 ✓
